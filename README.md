# Railway Flood Twin Project

Starter folder structure for the railway flood-risk digital twin MVP.

## Main folders
- `docs/` project documents
- `sql/` database schema and validation SQL
- `src/` Python scripts for ingestion, transformation, and validation
- `data/raw/` original source data
- `data/staging/` temporary transformed data
- `data/processed/` processed outputs
- `notebooks/` exploratory notebooks
- `outputs/` logs, exports, and figures
